## 5. Multilingual SEO Requirements

### 5.1 hreflang Implementation

Every page must include hreflang tags for all 3 locales plus x-default in the `<head>`:

```html
<link rel="alternate" hreflang="ms" href="https://roller-shutter-malaysia.vercel.app/ms/roller-shutter/kuala-lumpur" />
<link rel="alternate" hreflang="en" href="https://roller-shutter-malaysia.vercel.app/en/roller-shutter/kuala-lumpur" />
<link rel="alternate" hreflang="zh" href="https://roller-shutter-malaysia.vercel.app/zh/roller-shutter/kuala-lumpur" />
<link rel="alternate" hreflang="x-default" href="https://roller-shutter-malaysia.vercel.app/ms/roller-shutter/kuala-lumpur" />
```

**Rules:**
- `x-default` ALWAYS points to the `ms` version
- hreflang is bidirectional — if page A references page B, page B must reference page A
- hreflang must be present on ALL 156 pages (3 homepage + 150 location + 3 redirect)
- Use full absolute URLs (not relative)

**SEO reason:** hreflang tells Google which language version to serve to which user. x-default on ms ensures BM is the fallback for unmatched languages, matching our target market.

### 5.2 URL Structure

| Component | Pattern |
|---|---|
| Locale prefix | Always present: `/ms/`, `/en/`, `/zh/` |
| Product segment | `/roller-shutter/` (same across all locales — do NOT translate URL slugs) |
| Location slug | English slug format always: `kuala-lumpur` (same across all locales) |

**SEO reason:** Keeping URL slugs in English across all locales is a Google best practice. Translated URLs create duplicate content risk and complicate hreflang mapping.

### 5.3 Language-Specific Keyword Differences

| Concept | BM (primary) | EN | ZH |
|---|---|---|---|
| The product | roller shutter / pintu roller shutter / pintu besi gulung | roller shutter door | 卷帘门 / 铁卷门 |
| Install | pasang / pemasangan | install / installation | 安装 |
| Repair | baiki / repair | repair / fix | 维修 |
| Price | harga | price / cost | 价格 |
| Emergency | kecemasan / 24 jam | emergency / 24 hour | 紧急 / 24小时 |
| Free quote | sebut harga percuma | free quote / free quotation | 免费报价 |
| Near me | berdekatan / dekat sini | near me | 附近 |

**Key BM note:** Malaysian users commonly search in mixed BM/EN (e.g., "roller shutter murah" not "pintu besi gulung murah"). The content should use "roller shutter" as the primary term even in BM copy, with "pintu roller shutter" and "pintu besi gulung" as semantic variants.

### 5.4 Canonical Tags

- Each page's canonical URL points to itself (self-referencing canonical)
- No cross-language canonicals — each language version is its own canonical
- Homepage canonical: `https://roller-shutter-malaysia.vercel.app/[locale]`
- Location canonical: `https://roller-shutter-malaysia.vercel.app/[locale]/roller-shutter/[location]`

### 5.5 Sitemap

- Single sitemap at `/sitemap.xml`
- Must include all 156 pages with their full absolute URLs
- Each URL entry should include `<xhtml:link>` elements for all hreflang alternates
- Grouped by locale for clarity

### 5.6 robots.txt

```
User-agent: *
Allow: /
Disallow: /api/
Sitemap: https://roller-shutter-malaysia.vercel.app/sitemap.xml
```

**SEO reason:** Block API routes from crawling. Expose sitemap for discovery.
